# C Reserved Words

The set of reserved words below is used to build up the basic instructions of C; you can not use them in programs you write.

- A 
  - auto
- B  
  - break
- C
  - case
  - char
  - continue
- D
  - default
  - do
  - double
- E  
  - else
  - entry
  - extern
- F
  - float 
  - for 
- G
  - goto
- I
  - if
  - int
- L
  - long
- R
  - register
  - return
- S
  - short 
  - sizeof
  - static
  - struct
  - switch
- T
  - typedef
- U
  - union
  - unsigned
- V
  - void
  - volatile
- W
  - while

# In modern implementations:

- const
- enum 
- signed 
- void
- volatile
