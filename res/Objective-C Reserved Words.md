# Objective-C Reserved Words

Objective-C reserves certain words, so that you can’t, or shouldn’t, use them in your own code.

- A
  - auto
- B
  - break
- C  
  - case 
  - char
  - const
  - continue
- D
  - default
  - do
  - double
- E
  - else
  - enum
  - extern
- F
  - float 
  - for 
- G
  - goto
- I
  - if
  - inline
  - int 
- L
  - long
- R
  - register
  - restrict
  - return
- S
  - short
  - signed
  - sizeof
  - static
  - struct
  - switch
- T
  - typedef
- U
  - union
  - unsigned
- V
  - void
  - volatile
- W
  - while

- _Bool
- _Complex
- _Imaginary
